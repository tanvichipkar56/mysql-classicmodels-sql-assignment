# MySQL ClassicModels SQL Assignment

This repository contains my SQL assignment based on the **ClassicModels database**.  
The project demonstrates practical SQL queries used for analyzing business data.

## Topics Covered

- SELECT statements
- WHERE conditions
- AND / DISTINCT
- Wildcards using LIKE
- CASE statements for customer segmentation
- GROUP BY with aggregation functions
- HAVING clause
- Date functions
- Database constraints
- Creating tables and databases

## Dataset Used
ClassicModels sample database which simulates a **retail company selling scale models of classic cars**.

## Files Included

- classicmodels database.sql – Database structure
- MYSQL Assignment questions.docx – Assignment questions
- Tanvi SQL Assignment.sql – SQL query solutions

## Example Queries

- Fetch sales representatives reporting to a specific manager
- Identify product lines ending with "cars"
- Segment customers by region using CASE
- Find top selling products using GROUP BY and SUM
- Analyze monthly payment frequency
- Create tables using primary key, unique, and NOT NULL constraints
